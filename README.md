Activity 1 is the basic switch commands like encrypting our switch device and creating a management interface.

Activity 2 includes is a continuation of activity 1 but there's an intergration of router configurations which includes activating interfaces .
Activity 3 is the contiuation and improved version of activity 1 and 2. It includes 4 new switches each labeled uniquely and representing a different department.S3 represents that IT department,S4 representing HR department, S5 represents the finance department and S6 represents the sales department.

These four switches includes commands of vlans and each vlan has a pc that contains phone for VOICE vlan. S1 and S2 which were previously added in Activity 2 now contain trunk links to ensure that these different vlans or departments are able to communicate with each other .
